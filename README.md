# Sistema de Gestion Escolar

Sistema de gestion escolar completo en C++ con I/O estilo C.

## Funcionalidades

- Gestion de alumnos, profesores y materias
- Matriculas y asignacion de profesores
- Registro de notas y trabajos
- Control de asistencias
- Horarios y aulas
- Periodos academicos
- Usuarios con roles (Admin, Profesor, Alumno)
- Reportes de notas y asistencia

## Compilar y ejecutar

```
make
./bin/escuela
```

```

include/    Cabeceras
datos/      Archivos de datos (pipe-delimitados)
obj/        Objetos compilados (generado)
bin/        Ejecutable (generado)
```
